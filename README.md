# Task 2: Data Visualization and Storytelling

## Project Overview
This Power BI dashboard analyzes Superstore sales data to identify trends, profitability, customer segments, and regional performance.

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Monthly Sales Trend
- Category Analysis
- Segment Analysis
- Regional Performance
- Executive Summary

## Tools Used
- Power BI Desktop
- Microsoft Excel
- GitHub

## Key Insights
- Technology category generates highest sales.
- Consumer segment contributes most revenue.
- West region shows strongest performance.
- Some sub-categories require profitability improvement.
